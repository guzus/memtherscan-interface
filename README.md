# Memtherscan

## Useful Commands

Run on localhost

```
npm run start
```

Deploy

```
npm run deploy
```

## Infrastructure

- **Cloudflare R2**: Image, Video Storage
- **Firebase**: Database
- **GCP Cloud Run**: [API Server](https://github.com/guzus/memtherscan-server)
- **Github Actions, Github Pages**: [Frontend](https://github.com/guzus/memtherscan) Deployment
